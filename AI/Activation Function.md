
Step activation function:
 - 0 for x < alpha
 - 1 for x >= alpha

Trapezoid activation function:
-  0 for x < alpha
-  transfering from 0 to 1 linearly for x >= alpha and x < beta
-  1 for x >= beta

Sigmoid/logistic activation function:
- f(x) = 1/ (1 + e^(-x) )
- related to the logistics model (modelling population size)
